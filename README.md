# Leben in Saarbrücken

Eine moderne Website, die Saarbrücken als attraktiven Lebensort für junge, motivierte, engagierte und gebildete Menschen präsentiert.

## Projektbeschreibung

Diese Website wurde im Rahmen des 5. Assignments im Studienfach Online Marketing erstellt. Sie dient dazu, Saarbrücken als lebenswerte Stadt für die Zielgruppe zu bewerben und gleichzeitig den Studiengang Digital Media Marketing an der Hochschule Kaiserslautern zu promoten.

## Technische Umsetzung

### Technologien
- **HTML5** - Semantische Struktur
- **CSS3** - Modernes, responsives Design mit CSS Grid und Flexbox
- **JavaScript (Vanilla)** - Interaktivität und Event-Tracking
- **Google Fonts** - Inter Font Family

### Features
- ✅ Vollständig responsive Design (Mobile-First)
- ✅ Smooth Scrolling Navigation
- ✅ Intersection Observer für Scroll-Animationen
- ✅ Event-Tracking für Analytics (Google Analytics / GTM vorbereitet)
- ✅ SEO-optimierte Struktur
- ✅ Accessibility-Features
- ✅ Moderne UI/UX mit Gradienten und Animationen

## Installation & Nutzung

1. **Lokale Nutzung:**
   ```bash
   # Einfach die index.html in einem Browser öffnen
   open index.html
   ```

2. **Mit einem lokalen Server (empfohlen):**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (mit http-server)
   npx http-server
   
   # Dann im Browser öffnen: http://localhost:8000
   ```
